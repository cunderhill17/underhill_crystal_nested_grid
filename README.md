# Nested Grid Assignment
This repository is for the nested grid assignment in the Web Development 1 Course in IDP3 @ Fanshawe in which we have to demonstrate the use of CSS `Grid` and `Flexbox`, including `nested grids`, to organize content in a clear and structured way while also implementing `responsive images` and using the `order property` to manipulate the visual layout of elements.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open the `index.html` file in the browser of your choice.
